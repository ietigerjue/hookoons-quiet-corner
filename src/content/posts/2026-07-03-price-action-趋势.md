这是价格行为学的第二部分，趋势。本系列结构按照市场周期，市场周期的拆分（趋势，突破，通道，震荡区间），交易细节（信号K，缺口，买卖压，楔形等），交易执行（早盘，仓位管理等）进行记录。
 
## 趋势的定义
市场单一方向的运动，在一个包含200根K线的视图内，只应该有两段主要趋势。趋势通常以突破（大实体K线/窄通道开始）转换为同向的通道继续顺势运动。

## 日内交易中的趋势类型

| 日内交易中的趋势类型 | 描述                                |
| ---------- | --------------------------------- |
| 急速波动和通道    | 急速波动然后进入稳定的通道                     |
| 趋势性震荡区间    | 形成TR后经过一段小突破再次进入另一个TR             |
| 开盘即趋势      | 开盘大实体K线突破关键位置确定当日趋势               |
| 宽通道        | 期间有1-3Legs，逆势方开始制造连续的K线，动能交易者开始赚钱 |
| 小回调趋势      | 趋势中的回调K线数量少，实体小，体现了顺势方的急切         |
| 趋势延续       | PB不超过1st Leg的2/3，后继续2nd Leg       |
